- 500000 kbps represents 500/400mbps

- Debian
```
cd /usr/bin
wget -O limit "https://github.com/praiman99/Auto-Limit-Bandwith-VPS/raw/Beginner/limit-speed.sh"; chmod +x limit; bash limit
```
- Ubuntu
```
cd /usr/sbin
```
